# mfetch

A system information tool written in rust

# operating systems

Tested systems are

-   `Arch Linux`
-   `Debian`

# installation

#### ensure you have `cargo`, `rustup` and `rustc` installed

```bash
git clone https://github.com/GinoMoses/mfetch
cd mfetch
make all

```
